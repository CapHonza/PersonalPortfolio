Description

Custom Project Company is the very first website I ever created. It was developed as a final project in the Website Development course during the 1st semester of my Bachelor’s studies at Charles University.

The goal was to build a simple multi-page website of our choice using HTML & CSS with SVG graphics, focusing on clear structure, functional navigation and a consistent visual style.

This project gave me my first real experience with transforming an idea into a working website and helped me understand key principles of web structure, layout, and frontend development basics.

Technologies / Creation Process

The project was created using fundamental web technologies such as HTML, CSS, and SVG. The main focus was to build a multi-page website with a clear structure, internal navigation, and consistent styling.

During the development process, I worked on tasks like drawing shapes in SVG (e.g., a one-line house or simple flags), styling a chessboard layout using CSS, and practicing linking between pages to form a complete website.

I also worked with essential web resources including W3Schools, Icons8, and online color palette tools, while occasionally using AI assistance to support coding and design decisions.

What I Learned

As my very first real web project, this experience helped me understand how websites are built from scratch and how HTML, CSS and SVG technologies work together in practice.

I gained confidence in writing clean and structured code, working with navigation, linking pages, inserting images and improving visual hierarchy. I also experimented beyond the course requirements by trying new elements and exploring solutions independently.

Key Takeaways

  • Understanding the fundamentals of HTML & CSS
  • Learning about SVG and experimenting with Inkscape
  • Adding and styling graphics, icons and custom elements
  • Building a multi-page project with consistent navigation
  • Improving visual structure and basic responsiveness
  • Developing problem-solving skills and independence

Looking back, I’m proud of this project — it was a successful first step into the world of web development and a strong base for future growth.
