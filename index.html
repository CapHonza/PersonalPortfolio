Ahoj! Jsem Honza, student oboru Informační technologie na Pedagogické fakultě Univerzity Karlovy, kde po prázdninách nastupuji do třetího ročníku. 🎓 Mým hlavním zaměřením a absolutní prioritou je web development. Tvořím weby pomocí HTML, CSS a naplno se nořím do možností JavaScriptu. 🌐 Mým cílem je stavět interaktivní webové aplikace a výhledově ovládnout technologie jako React, Node.js nebo p5.js. 💻 Baví mě minimalismus, čistý kód a dotahování věcí do konce. Jsem spolehlivý, zakládám si na upřímné a přátelské komunikaci a plynule mluvím anglicky (B2/C1). 🌍
