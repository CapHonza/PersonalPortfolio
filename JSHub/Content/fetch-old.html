Klasický Fetch a práce s Promisy

Každý vývojář by měl znát základy. V této ukázce z mého tréninku na platformě Codedex pracuji s klasickým zápisem Fetch API pomocí zřetězených metod .then().

Skript posílá požadavek na externí API pro generování vtipů. Hlavní výzvou zde bylo správně zachytit odpověď, převést ji do formátu JSON a následně data vypsat do DOMu. Ačkoliv je tento přístup plně funkční, při složitějších operacích může vést k nepřehlednému kódu (tzv. callback hell).
